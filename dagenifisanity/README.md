# Sanity Clean Content Studio

Congratulations, you have now installed the Sanity Content Studio, an open source real-time content editing environment connected to the Sanity backend.

Now you can do the following things:

-   [Read “getting started” in the docs](https://www.sanity.io/docs/introduction/getting-started?utm_source=readme)
-   [Join the community Slack](https://slack.sanity.io/?utm_source=readme)
-   [Extend and build plugins](https://www.sanity.io/docs/content-studio/extending?utm_source=readme)

# How to:

1. enter 'npm install -g @sanity/cli' in terminal to install the sanity client
2. enter 'sanity install' to install the studio dependencies
3. enter 'sanity start' to run the studio, which will open at localhost:3333
